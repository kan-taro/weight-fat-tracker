# Weight & Body Fat Tracker

体重・体脂肪率を手軽に記録し、推移を可視化できる **ブラウザだけで完結する Web アプリ**です。  
Tailwind CSS と Chart.js を CDN から読み込み、ローカルストレージにデータを保存しますので、サーバーやデータベースは不要です。

---

## 📌 主な特徴

| # | 内容 |
|---|------|
| 1️⃣ | **日付・体重(kg)・体脂肪率(%)** を入力して登録。 |
| 2️⃣ | データはブラウザの **ローカルストレージ** (`weight_fat_records`) に永続化（ブラウザを閉じても残ります）。 |
| 3️⃣ | テーブルで一覧表示、Chart.js で折れ線グラフ化。 |
| 4️⃣ | CSV のインポート/エクスポート機能。 |
| 5️⃣ | 「全削除」ボタンで一括消去（確認ダイアログ付き）。 |

> **⚠️** 本アプリは「静的サイト」です。サーバー側の処理や認証は実装されていません。

---

## 🛠 技術スタック

## 🚀 Quick Start


1. リポジトリをクローン
  - git clone https://github.com/kan-taro/weight-fat-tracker.git
  - cd weight-fat-tracker
  - open index.html
  - お使いのブラウザで表示されます。お気に入りに登録すればブラウザから直接開けます。


## 使い方

1. **データ登録**  
   - フォームに「日付」「体重 (kg)」「体脂肪率 (%)」を入力 → 「登録」ボタン。  

2. **個別編集/削除**  
   - テーブル内の「編集」または「削除」ボタンをクリック。  

3. **全データ消去**  
   - 「全削除」ボタン（確認ダイアログ付き）を押すとローカルストレージがクリアされます。  

4. **CSV エクスポート**  
   - 「CSV出力」ボタンで現在のレコードを CSV ファイルとしてダウンロード。  

5. **CSV インポート**  
   - 「CSVインポート」→ファイル選択。既存の日付は上書きされます。  

> データはブラウザのローカルストレージに保存され、サーバーには送信されません。
